Digital Literacy & Creative Coding Mentor | 2022-2024

This repository contains a collection of interactive sketches and digital art used during my time as a Digital Literacy & Creative Coding Mentor at Ohlone College. My workshops focused on teaching students how to use p5.js to explore the intersection of mathematics, data science, and visual design.
[Interactive Preview]

01 / Perlin Noise Nebula

Concept: Simulating fluid dynamics and stellar gas clouds using 2D Perlin noise with interactive color mapping and particle flow systems.

Learning Objective: Understanding non-linear randomness, vector fields, and how natural phenomena can be represented through mathematical noise functions.

02 / Algorithmic Symmetry

Concept: A generative art piece utilizing trigonometric identities to create complex geometric forms through recursive pattern generation and harmonic oscillation.

Learning Objective: Mapping mathematical functions to visual properties (color, stroke, position) and understanding symmetry groups in computational design.

03 / Data-Driven Particle Systems

Concept: An interactive system where particle behavior is dictated by real-time data inputs, demonstrating emergent complexity from simple rules.

Learning Objective: Integrating Data Science principles with real-time rendering, understanding object-oriented programming, and creating responsive visual systems.
